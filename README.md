# q2a-pmplus

<b>Plugin Name:</b>  PM Plus <br>
<b>Price:</b> 30 USD (contact me to buy) <br>
<b>Plugin Description:</b> This is a premium-paid Q2A plugin that re-structures private messages (PM) style like facebook message conversation and it also gives user blocking/unblocking feature to send PMs.<br>
<br>
<ul class="first">
	<b>Plugin Functionality:</b>
	<li>Plugin creates 2 tables at database to store data that plugin will use.</li>
	<li>New style message conversation is almost similar like Facebook messaging.</li>
	<li>With the new style it is very easy to follow-up message conversations.</li>
	<li>The plugin is fully integrated with Q2A on-site notification plugin.</li>
	<li>Users can block any other users from sending and receiving PM's. If User-A blocks User-B, then User-B cannot send PM to User-A, and as User-A blocked User-B, User-A also cannot send PM to User-B.</li>
	<li>Users can anytime unblock/remove PM blocks.</li>
	<li>The Q2A Message Page is almost completely re-structured. 3 sub-navigation tabs are added: Message Box, Who I Blocked, Who Blocked Me</li>
	<li>Tab Message Box displays the conversation list of users with logged in user.</li>
	<li>Tab Who I Blocked displays list of users who you have blocked and list count.</li>
	<li>Tab Who Blocked Me displays list of users who have blocked you and list count.</li>
	<li>Plugin also adds quick PM button on each post (question, answer, comment, reply). When clicked to quick PM button then pop up will appear where post-id will be already written. It is good as it will show to what post the message is related. For instance, if you have posted an answer (id-5), on that answers buttons there will be PM button, so any users can click on it and quickly PM you regarding related answer as ID will be included in the PM message.</li>
	<li>There is no any core hacks, it is 100% pluginarized.</li>
